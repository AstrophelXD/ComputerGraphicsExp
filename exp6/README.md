# 实验6：曲面体真实感显示（建模与消隐）

本目录代码依据 `docs/图形实验六.docx` 要求实现，并对齐 `docs/第9章  建模与消隐.pdf` 中的建模/消隐思路。
当前版本按你的要求仅实现 **21组及以后** 对应内容，且代码为精简版（完全基于 MFC）：

- 显示圆环体（Torus）；
- 使用绿宝石材质（Emerald）；
- 使用单光源模型设置环境光和点光源强度：
  - `Iar=Iag=Iab=200`
  - `Ipr=Ipg=Ipb=250`
- 曲面体不直接调用现成实心体，而是按参数方程离散成三角形/四边形网格：
  - 圆柱（9-3）
  - 圆锥（9-4）
  - 圆环（9-5）
- 使用深度缓冲（Z-Buffer）与背面剔除实现隐面消除。

材质参数采用课程常见表（表 10-3）中的 Gold / Ruby / Emerald 系数。

## 文件说明

- `main.cpp`：MFC + OpenGL 实现，采用 `CFrameWnd + CView` 架构，核心绘制在 `CView::OnDraw`。

## 编译运行（MFC）

本版本改为 **MFC 窗口绘图**（不再使用 GLUT 主窗口循环），推荐使用 Visual Studio + MFC 组件编译。
先确认 VS 已安装组件：**C++ MFC for latest v143 build tools (x86 & x64)**。

程序启动后直接绘制“圆环体 + 绿宝石材质”（无需组号分支）。

### 快速构建

在仓库根目录双击 `build_mfc.bat`，会调用 VS 开发者命令行并编译 `exp6/exp6.exe`。

## 附录：`main.cpp` 原理详解（按模块）

本附录按模块解释 `exp6/main.cpp` 如何完成“圆环体 + 绿宝石材质 + 消隐”的实时绘制。

### 1) 依赖与数据结构模块

- `afxwin.h`：MFC 应用/窗口/视图框架；
- `gl/GL.h`、`gl/GLU.h`：固定管线 OpenGL 与投影/相机辅助函数；
- `Vec3`：三维向量结构（`x/y/z`）；
- `PI`：圆环参数方程角度步进常量；
- `g_angle`：全局旋转角，用于动画。

该模块的目标是提供最小必要运行时支撑，不引入额外框架或复杂抽象。

### 2) 向量工具模块（几何基础）

- `sub(a,b)`：向量减法，用于构造边向量；
- `cross(a,b)`：叉积，得到面法线方向；
- `normalize(v)`：单位化法线，保证光照计算稳定。

这里的法线计算采用“两个边向量叉积”的标准方式。若长度过小，回退到默认法线，避免数值异常。

### 3) 四边形发射模块：`emitQuad(...)`

功能：把一个参数网格单元转成 OpenGL 顶点输出。

流程：

1. 用 `v0/v1/v2` 计算面法线；
2. `glNormal3f` 提交法线；
3. 顺序提交四个顶点（`glVertex3f`）。

由于本实现采用固定管线光照，法线方向和顶点顺序直接影响明暗与背面剔除结果。

### 4) 圆环参数建模模块：`drawTorus(...)`

功能：按参数方程离散圆环，并用 `GL_QUADS` 绘制。

关键参数：

- `r1`：主半径（圆环中心线半径）；
- `r2`：管半径（截面小圆半径）；
- `nAlpha`：绕主圆离散数；
- `nBeta`：截面离散数。

核心思想：

- 双重循环遍历参数域 `(alpha, beta)`；
- 每个网格单元生成 4 个顶点；
- 顶点坐标由参数方程直接计算；
- 调用 `emitQuad` 统一处理法线和顶点提交。

这样实现满足“曲面离散网格化”要求，不依赖现成实心体函数。

### 5) 光照材质与消隐状态模块：`setupLightingAndMaterial()`

本模块在 OpenGL 上下文初始化后调用一次，配置渲染状态：

- 光照：
  - 开启 `GL_LIGHTING` 和 `GL_LIGHT0`；
  - 设置环境光、漫反射光、镜面光强度（对应实验给定强度）；
  - 设置点光源位置 `lightPos`。
- 材质：
  - 设置 Emerald 的环境/漫反射/镜面参数；
  - 设置高光指数 `shininess=76.8`。
- 消隐：
  - 开启深度测试 `GL_DEPTH_TEST`（Z-Buffer）；
  - 开启背面剔除 `GL_CULL_FACE`，并指定正面绕序为 `GL_CCW`。

这部分共同决定“真实感 + 隐面消除”的最终视觉效果。

### 6) OpenGL 上下文生命周期模块（`CExp6View`）

`CExp6View` 中 OpenGL 生命周期集中管理：

- `OnCreate`：
  - 选择像素格式（RGBA + 深度缓冲 + 双缓冲）；
  - 创建并绑定 `HGLRC`；
  - 初始化光照材质状态；
  - 启动定时器驱动动画。
- `OnDestroy`：
  - 停止定时器；
  - 解绑并销毁 `HGLRC`；
  - 释放资源。

这样可确保渲染资源与窗口生命周期一致，避免上下文泄漏。

### 7) 视口与投影矩阵模块：`OnSize(...)`

窗口尺寸变化时：

- 更新 `glViewport` 适配新窗口大小；
- 切到 `GL_PROJECTION` 并重建透视投影 `gluPerspective`；
- 近远裁剪面为 `0.1 ~ 100.0`；
- 切回 `GL_MODELVIEW`，供后续模型变换使用。

该模块保证窗口缩放后画面比例与透视关系保持正确。

### 8) 动画驱动模块：`OnTimer(...)`

- 每帧（约 16ms）把 `g_angle` 增加 `0.6` 度；
- 使用 `fmod` 将角度限制在 `0~360`；
- 调用 `Invalidate(FALSE)` 触发重绘。

这实现了绕 Y 轴连续旋转的动态观察效果。

### 9) 主渲染流程模块：`OnDraw(CDC*)`

每次重绘执行以下步骤：

1. 绑定当前 OpenGL 上下文；
2. 清屏（颜色缓冲 + 深度缓冲）；
3. 设置观察矩阵：
   - `gluLookAt` 指定相机位置与观察中心；
   - 额外 `glRotatef` 调整俯仰与自转；
4. 调用 `drawTorus(1.1, 0.4, 64, 32)` 绘制离散圆环；
5. `SwapBuffers` 双缓冲交换显示。

最终形成平滑旋转、带高光与隐面消除的圆环体。

### 10) MFC 外壳模块（`Frame` / `App`）

- `CExp6Frame`：创建主窗口与标题；
- `CExp6App::InitInstance()`：
  - 初始化应用；
  - 创建 `Frame` 和 `View`；
  - 挂接视图并显示窗口。

这一层负责窗口启动与消息循环，不参与具体图形算法，实现上保持最小化。
