# 实验6：曲面体真实感显示（建模与消隐）

本目录代码依据 `docs/图形实验六.docx` 要求实现，并对齐 `docs/第9章  建模与消隐.pdf` 中的建模/消隐思路。
当前版本按你的要求仅实现 **21组及以后** 对应内容，且代码为精简版（完全基于 MFC）：

- 显示圆环体（Torus）；
- 使用绿宝石材质（Emerald）；
- 使用单光源模型设置环境光和点光源强度：
  - `Iar=Iag=Iab=200`
  - `Ipr=Ipg=Ipb=250`
- 曲面体不直接调用现成实心体，而是按参数方程离散成三角形/四边形网格：
  - 圆柱（9-3）
  - 圆锥（9-4）
  - 圆环（9-5）
- 使用深度缓冲（Z-Buffer）与背面剔除实现隐面消除。

材质参数采用课程常见表（表 10-3）中的 Gold / Ruby / Emerald 系数。

## 文件说明

- `main.cpp`：MFC + OpenGL 实现，采用 `CFrameWnd + CView` 架构，核心绘制在 `CView::OnDraw`。

## 编译运行（MFC）

本版本改为 **MFC 窗口绘图**（不再使用 GLUT 主窗口循环），推荐使用 Visual Studio + MFC 组件编译。
先确认 VS 已安装组件：**C++ MFC for latest v143 build tools (x86 & x64)**。

程序启动后直接绘制“圆环体 + 绿宝石材质”（无需组号分支）。

### 快速构建

在仓库根目录双击 `build_mfc.bat`，会调用 VS 开发者命令行并编译 `exp6/exp6.exe`。
